BatchApplySubstance script README
----------------------------------------

Putting this script in your project assets folder will allow you to apply your Procedural Materials
to any number of meshes in one click.

INSTALLATION AND USAGE
----------------------------------------
1. Copy the script in an "editor" folder inside you project's assets folder.
2. Click on Window > BatchApplySubstance to bring up the selection window.
3. Select the Procedural Material you want to apply in the Substance Mat. slot, then select all the 
objects you want to apply the material to in your Hierarchy window.
4. Hit Apply and you're done :)
